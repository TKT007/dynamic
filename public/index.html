❓ Como Usar - Guia Completo

🎯 Para que serve?

Este sistema permite criar sites automaticamente na Vercel que executam códigos especiais APENAS quando visitantes vêm do TikTok através de links com parâmetros específicos!

1️⃣ Configure Token

• Acesse vercel.com/account/tokens
• Crie um novo token
• Cole na aba "Configuração"
• Clique "Testar Conexão"

2️⃣ Escolha Template

• Landing Page: Página simples
• E-commerce: Loja básica
• Upload: Seus arquivos
• Suporte: HTML, CSS, JS, imagens
